# Title

# Comprehensive Whole-Genome and Structural Analysis of Acinetobacter baumannii SRR5891469: Antimicrobial Resistance, Virulence Factors, and OXA-23 Mutation Characterization

## Project Overview

This project presents a comprehensive genomic and structural analysis of Acinetobacter baumannii isolate SRR5891469 using whole-genome sequencing data obtained from the NCBI Sequence Read Archive (SRA).

The study focused on identifying antimicrobial resistance determinants, virulence-associated genes, mutations within the OXA-23 beta-lactamase gene, and evaluating the structural implications of these mutations through computational protein modeling and validation.

## Objectives

* Perform whole-genome analysis of Acinetobacter baumannii
* Identify antimicrobial resistance genes
* Detect virulence-associated factors
* Characterize mutations in the OXA-23 beta-lactamase gene
* Predict and validate the OXA-23 protein structure
* Integrate genomic and structural findings

## Bioinformatics Tools

FastQC, FastP, BWA-MEM, SAMtools, BCFtools, Prokka, ABRicate, IGV, SWISS-MODEL, PROCHECK, ERRAT, Verify3D

## Key Findings

* 3790 annotated genes identified
* 3772 protein-coding genes
* Multiple antimicrobial resistance determinants detected
* 110 virulence-associated genes identified
* OXA-23 mutations characterized
* High-quality protein structure validated
